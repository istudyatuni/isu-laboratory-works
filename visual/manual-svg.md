# Элементы

**Прямоугольник**

```xml
<rect x="" y="" width="" height="" rx="" ry="" />
```

`x`, `y` - координаты верхнего левого угла

`width`, `height`

`rx`, `ry` - скругление углов

**Общие параметры**

`fill` - заливка (цвет или `none`)

`stroke` - цвет контура

`stroke-width` - ширина контура

**Окружность**

```xml
<circle cx="" cy="" r="" />
```

`cx`, `cy` - центр окружности

`r` - радиус
